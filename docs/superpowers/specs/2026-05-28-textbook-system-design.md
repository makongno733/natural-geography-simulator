# 中学地理教学系统 — 教材模块设计方案

## 概述

在现有 Vue 3 + Three.js 自然地理教学应用基础上，新增人教版教材内容浏览系统，面向初中生和高中生。覆盖初中 4 册 + 高中 5 册 人教版地理教材，按目录逐章逐节展示文字内容，并为未来 3D 互动教学预留接口。

## 整体架构

### 入口整合

`App.vue` 改造为总入口，替换 `natural-geo-system.html` 的角色。

- 保留大标题「中学地理教学系统」白底红字 Ma Shan Zheng 行书风格
- 顶部 Tab 切换：「📖 课本」|「🔧 其他」
- 「课本」为主入口，显示学段选择卡片（初中/高中）
- 「其他」收纳原大气结构、地貌沙箱模块，以灰色卡片弱化展示
- 大标题 UI 后续再推送上线，初期聚焦内容框架

### Hash 路由

```
#/                                              首页（课本 Tab 默认）
#/textbook                                      学段选择页
#/textbook/:grade                               显示该学段的教材列表
#/textbook/:grade/:book                         显示该册的章列表
#/textbook/:grade/:book/:chapter                显示该章的节列表
#/textbook/:grade/:book/:chapter/:section       节内容页
#/atmosphere                                    大气结构
#/sandbox                                       地貌沙箱
```

## 导航设计

多级卡片式导航，从顶层到底层：

```
首页 → 选初中/高中 → 选教材册 → 选章 → 节内容页
```

每级均为卡片网格布局，点击进入下一级。节内容页左侧显示当前册的章节目录侧栏（可快速跳转），右侧为内容区。

## 数据组织

### 教材目录（人教版）

**初中（4 册）：**
- 七年级上册（2024 秋版）
- 七年级下册（2025 春版）
- 八年级上册（2025 秋版）
- 八年级下册（2026 春版）

**高中（5 册）：**
- 必修 第一册
- 必修 第二册
- 选择性必修 1 — 自然地理基础
- 选择性必修 2 — 区域发展
- 选择性必修 3 — 资源、环境与国家安全

### 内容存储格式

每节内容存为 JSON 文件，结构：

```json
{
  "id": "ch01-sec01",
  "grade": "初中",
  "book": "七年级上册",
  "chapter": "第一章",
  "chapterTitle": "地球",
  "section": "第一节",
  "sectionTitle": "地球的形状和大小",
  "content": {
    "keyPoints": ["知识点 1", "知识点 2"],
    "body": "详细讲解文字（支持 HTML）",
    "images": [],
    "interactive": null
  },
  "prev": "ch01-sec00",
  "next": "ch01-sec02"
}
```

### 初始填充策略

第一版仅搭建完整目录框架，节内容以占位文字填充。后续从 PDF 提取文字逐步填充。

## 现有模块处理

「其他」Tab 下展示两个入口卡片（灰色弱化）：
- **大气垂直结构与化学组成** — `#/atmosphere`
- **地貌演化与河流类型** — `#/sandbox`

功能与现有实现完全一致，不进行改造。

## 3D 预留

章节内容页的 `content.interactive` 字段保留为 `null`，后续可引用 Three.js 场景组件。3D 交互组件复用 `src/lib/` 下的现有场景逻辑，或针对特定章节新建场景文件。
